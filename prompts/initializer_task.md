# Initializer Task

You are starting a fresh project. The project directory is: `{{PROJECT_DIR}}`

## Steps

1. Read `task_spec.txt` in the project directory to understand what needs to be built
2. Create `TASKS.md` in the project directory with:
   - A brief summary of the project
   - A numbered task list using `- [ ]` checkboxes, broken into small implementable steps (aim for 8-15 tasks)
3. Initialize the project structure (create necessary folders, package.json, etc.)
4. Implement the first task from TASKS.md
5. Mark the first task complete in TASKS.md
6. Report what you have done and what comes next

Keep tasks small enough that each one can be completed in a single session.
